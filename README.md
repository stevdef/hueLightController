# Philips Hue Lights Controller
Control Philips Hue lights with JavaScript (Hue Bridge device needed)

## Main functions
lightsStatus() 
  --> for each light, display the state, brightness, color

setLightsOnOrOff
  --> Lights on of Lights Off

setBri()
  --> set Brightness

setColor()
  --> set light color
